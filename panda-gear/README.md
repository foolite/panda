Panda Gear
==========

A Lightweight Java MVC Web Framework


Features
------------

 - panda.aop: AOP library
 - panda.asm: Java ASM library, used by AOP
 - panda.mvc: MVC web framework
 - panda.servlet: some useful classes for J2EE servlet
