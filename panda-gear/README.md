Panda Gear
==========

A Lightweight Java MVC Web Framework


Features
------------

 | Package                | Description                                                               |
 |------------------      |---------------------------------------------------------------------------|
 | panda.ioc              | IOC library                                                               |
 | panda.mvc              | MVC web framework                                                         |
 | panda.servlet          | useful classes for J2EE servlet                                           |
