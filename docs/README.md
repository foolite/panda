 Panda Framework
=================

Panda is a Java development framework, it can help you quickly create a WEB application.

 - [English Readme](Readme_en)



Panda是一个Java的开发框架，它可以帮你快速的建立一个WEB应用。

 - [中文说明](Readme_zh)



## Requirements
 - Java 7
