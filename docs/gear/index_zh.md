 Panda Gear
=================

Panda Gear是Panda开发框架中关于Web应用(Servlet)的一个模块，类似Spring-MVC, Struts2，包含了一些常用的Taglib。


Panda Gear由以下几个部分组成。

 | 模块                      | 说明                                                                      |
 |---------------------------|---------------------------------------------------------------------------|
 | [IOC](ioc/ioc_zh.md)     | IOC模块，类似Spring。   |
 | [MVC](mvc/mvc_zh.md)     | MVC模块。类似Spring-MVC, Struts2。包含了一些常用的Taglib。 |
