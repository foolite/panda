 Panda Framework
=================

Panda is a Java development framework, it can help you quickly create a WEB application.

Pandaフレームワークは企業むけ高品質な Java Web システムを効率的に開発するためのフレームワークである。
設計、開発、保守全ての開発フェースに対し、開発コストを削減、品質を向上するために設計されています。
